Brief

Five homepage directions, built around new profile cards.

The live profile page now carries a lot more signal than the homepage cards can express. These five directions each start from a new profile card design that uses the new data — then build a homepage around that card's aesthetic logic. Same three sample operators in each direction so you can compare apples to apples.

New data in every card
3 activity heatmaps (sessions · tokens · est. cost), plugin-source-colored workflow nodes, skill chips with icons, hero per-week stats, autonomy class, model split.
Shared operators
Kalid (orchestrator) · Julia (craftsperson) · Marcus (automator). Same numbers, same skills, rendered five different ways.
Directions
1 editorial · 2 tactical · 3 collectible · 4 dense · 5 poster. Each homepage's layout follows from its card's visual logic.
Note: Each direction shows a detail zoom of the card first, then the full homepage containing that card style. Feedback textareas are at the bottom of every direction.
