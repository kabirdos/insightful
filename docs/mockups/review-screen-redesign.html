Combined Review Screen

Merges the old Redact + Preview steps into a single "Review" screen. The user sees exactly what their published profile will look like, with inline controls to hide sections, redact sensitive items, and toggle individual stats. This IS the preview.

Key change: No more separate "Redact" and "Preview" steps. This single screen shows the profile as visitors will see it, with edit controls overlaid. Every section has a visibility toggle; every sensitive item has inline Redact/Alias/Keep buttons.
