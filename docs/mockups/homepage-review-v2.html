Design Context & Decisions

Key insights from the 10-agent consensus analysis and your round 1 feedback that shaped these directions.

Lead with real data (8/10 agents)

The sample report IS the hero. Show actual developer profiles before asking for anything.

Token heatmap as hook (your feedback)

Tokens are the currency devs talk about. The heatmap both visualizes this and promotes /harness-profile for richer data.

Browse-first CTA (6/10 agents)

Social proof before ask. Let visitors see value before committing to upload.

Light mode exploration (your feedback)

Dark mode is standard for dev tools but you're not sure it's right here. Two light options (E, G) and one warm dark (F) to compare.

Compact density (your feedback)

Showcase was too spread out. All three directions use tighter spacing and horizontal layouts for stat rows.

/harness-profile as upgrade path

Each mockup mentions the harness-profile skill as a way to get deeper data (token usage, cost analysis). This creates a funnel from /insights → /harness-profile.
