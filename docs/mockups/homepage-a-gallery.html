How it works

What is /insights?

Claude Code has a built-in command that analyzes your usage patterns, tool preferences, and workflow habits across all your projects. Insight Harness lets you share that analysis with the community.

1
Run the command. Type claude /insights in your terminal to generate your report.
2
Upload your report. Paste the output or drag the HTML file into Insight Harness.
3
Join the gallery. Your report appears in the public gallery. Browse others, compare workflows, and learn new techniques.
Terminal
$ claude /insights

# Analyzing your Claude Code usage...
# Scanning 847 sessions across 45 projects

--- Your Claude Code Insights ---
Sessions: 847
Messages: 12,431
Top tools: Edit, Bash, Read
Features: parallel agents, worktrees
Style: "Delegation-heavy planner"

Report saved to ./insights-report.html
